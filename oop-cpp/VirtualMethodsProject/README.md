Задание: реализация иерархии классов с виртуальным методом расчёта площади:
- **Figure** — базовый абстрактный класс.
- **Rectangle**, **Circle**, **Triangle** — наследники с переопределением `GetArea()`.

### Файлы проекта:
- `include/Figure.h`
- `include/Rectangle.h`
- `include/Circle.h`
- `include/Triangle.h`
- `main.cpp`
